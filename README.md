Automação de E-mail em Java

Este projeto é uma automação simples para envio de e-mails utilizando Java.
Inicialmente, realizei esse mesmo projeto em Python, o que me despertou o interesse em recriá-lo também em Java. A ideia foi comparar as duas linguagens, entender as diferenças na implementação e fortalecer meus conhecimentos na linguagem Java.

O projeto utiliza bibliotecas tradicionais como javax.mail e javax.activation para configurar o servidor SMTP, autenticação e envio de mensagens.

Funcionalidades

Envio de e-mail usando Gmail SMTP.

Autenticação por senha de app.

Configuração simples e direta no código.

Tecnologias utilizadas

Java

javax.mail 1.6.2

javax.activation 1.2.0

Como executar

Certifique-se de ter o Java instalado.

Importe o projeto no IntelliJ IDEA ou VS Code.

Adicione as dependências presentes na pasta lib.

Execute a classe EmailAutomation.java.

English 
Email Automation in Java

This project is a simple automation for sending emails using Java.
I originally created this project in Python, and I enjoyed the experience, so I decided to reproduce it in Java as well. The goal was to compare both languages, understand the differences in implementation, and strengthen my Java programming skills.

The project uses traditional libraries such as javax.mail and javax.activation to handle SMTP configuration, authentication, and message sending.

Features

Send email using Gmail SMTP.

App password authentication.

Direct and easy configuration in the code.

Technologies Used

Java

javax.mail 1.6.2

javax.activation 1.2.0

How to Run

Make sure you have Java installed.

Import the project into IntelliJ IDEA or VS Code.

Add the dependencies located in the lib folder.

Run the EmailAutomation.java class.
